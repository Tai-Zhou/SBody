class Integrator

A wrapper of the gsl_odeiv2_evolve.

Integrator(int (*function)(double, const double*, double*, void*), int (*jacobian)(double, const double*, double*, double*, void*), int coordinate, void *params = nullptr, const gsl_odeiv2_step_type *type = gsl_odeiv2_step_rk8pd)

Construct a new Integrator object.

Parameters:

  • function â The function calculates

    \[\frac{\mathrm dy_i(t)}{\mathrm dt}=f_i\left[t,y_1(t),\dots,y_n(t)\right].\]

  • jacobian â The jacobian of the function

    \[J_{ij}=\frac{\partial f_i\left[t,y(t)\right]}{\partial y_j}.\]

  • coordinate â The coordinate of the system, 0 for Cartesian, 1 for spherical, and 2 for modified spherical. The modified spherical coordiante maps the polar angle from \(\theta\in[0,\pi]\) to

    \[\begin{split} \theta'\equiv\begin{cases}\theta & (\theta\leq\pi/2)\\ \theta-\pi & (\theta>\pi/2)\end{cases}. \end{split}\]

  • params â The parameters passed to the function, like the PN parameter, or the spin of the black hole.

  • type â Type of the algorithms. Explicit embedded Runge-Kutta Prince-Dormand (8, 9) method is set by default.
