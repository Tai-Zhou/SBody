Construct a new Integrator object.

Parameters:

  • function ‚Äì The function calculates

    \[\frac{\mathrm dy_i(t)}{\mathrm dt}=f_i\left[t,y_1(t),\dots,y_n(t)\right].\]

  • jacobian ‚Äì The jacobian of the function

    \[J_{ij}=\frac{\partial f_i\left[t,y(t)\right]}{\partial y_j}.\]

  • params ‚Äì The parameters passed to the function, like the PN parameter, or the spin of the black hole.

  • type ‚Äì Type of the algorithms. Explicit embedded Runge-Kutta Prince-Dormand (8, 9) method is set by default.

template<typename Type>
Type SBody::Flux(Type luminosity, Type magnification, Type redshift)

Calculate the bolometric flux. The result is.

\[F_\text{obs}=\frac{L_\text{src}}{(1+z)^2}\frac{\Omega_\text{src}}{\Omega_\text{obs}}.\]
The additional \((1+z)^{-1}\) comes from the expansion of the frequency that \((1+z)^{-1}=\frac{\nu_\text{obs}}{\nu_\text{src}}\).

Parameters:

  • luminosity ‚Äì intrinsic luminosity of the source, \(L_\text{src}\)

  • magnification ‚Äì magnification of the source, including the relativistic redshift and beaming, and gravitational lensing. \(\frac{\Omega_\text{src}}{(1+z)\Omega_\text{obs}}\)

  • redshift ‚Äì relativistic redshift of the source, \(1+z\)

Returns:

result
